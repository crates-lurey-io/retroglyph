## 0.1.0 - Initial release

- Initial public release. GPU rendering backend for retroglyph: native OpenGL 3.3 core and browser
  WebGL2 from a single `glow` codebase. One instanced draw call per frame (a unit quad instanced
  `cols * rows` times) sampling an `R8` glyph-atlas `TEXTURE_2D_ARRAY`, with per-cell
  foreground/background color. Implements `retroglyph_window::Presenter`, so it drops into the same
  winit windowing loop as `retroglyph-software`.
